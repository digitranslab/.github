### Migration Strategies (6Rs)

| Strategy | Description | Best For |
|----------|-------------|----------|
| **Rehost** | Lift and shift to cloud | Quick migrations, legacy systems |
| **Replatform** | Minor optimisations during migration | Database migrations, containerisation |
| **Refactor** | Re-architect for cloud-native | High-value applications |
| **Repurchase** | Move to SaaS solutions | Commodity workloads |
| **Retire** | Decommission unused systems | Legacy applications |
| **Retain** | Keep on-premises | Compliance-restricted workloads |
